# 🤖 News Agent (ニュース・エージェント)

毎朝7時、**世界中の最新ニュースから「あなたに必要な情報だけ」をAIが厳選し、Discordに届けてくれる** 自分専用のAIエージェントです。

PCは不要。一度設定すれば、**完全無料・全自動** で動き続けます。

---

## ✨ 何ができるの？
*   指定したキーワード（例: "AI", "デザイン", "ポケモン"）に関する最新記事を自動収集
*   **Google Gemini 3 Flash (最新の高速AI)** が中身を読み、「重要かどうか」を判断
*   重要な記事だけを日本語で要約し、Discordに通知
*   **完全無料**（GitHub ActionsとGemini無料枠を使用）

---

## 🚀 導入手順（所要時間：約10分）

プログラミングの知識は不要です。以下の手順通りに進めてください。

### Step 1. この機能をあなたのGitHubにコピーする

ページ右上の緑色のボタン **「Use this template」** → **「Create a new repository」** をクリックします。

> **📷 [ここに「Use this template」ボタンのスクリーンショットを入れる]**
> *   Repository name: 好きな名前を入力（例: `my-news-bot`）
> *   Public/Private: **「Private（自分だけ・非公開）」** を推奨します。
> *   最後に「Create repository」を押す。

これで、あなたのGitHubアカウントにこのツールがコピーされました！

### Step 2. 必要な「鍵」を2つ用意する

このボットを動かすには、以下の2つの「鍵（URLとパスワード）」が必要です。メモ帳などに控えておいてください。

**1. Discord Webhook URL**（ニュースを届ける場所）
*   通知を送りたいDiscordチャンネルの「設定（歯車）」→「連携サービス」→「ウェブフック」へ。
*   「新しいウェブフック」を作成し、**「ウェブフックURLをコピー」** します。
> **📷 [ここにDiscordのWebhook URLコピー画面のスクショを入れる]**

**2. Google Gemini APIキー**（AIの頭脳を使うパスワード）
*   [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセス。
*   Googleアカウントでログインし、**「Create API key」** をクリックしてキーを発行・コピーします。
> **📷 [ここにGoogle AI Studioのキー発行画面のスクショを入れる]**

### Step 3. 鍵をGitHubに登録する

コピーしたリポジトリのページに戻り、設定を行います。

1.  画面上のタブ **「Settings」** をクリック。
2.  左メニューの **「Secrets and variables」** → **「Actions」** をクリック。
3.  **「New repository secret」** （緑のボタン）をクリックして、以下の2つを登録します。

> **📷 [ここにSecrets登録画面のスクショを入れる]**

| Name（名前） | Secret（中身） |
| :--- | :--- |
| **`DISCORD`** | Step 2でコピーした **Discord Webhook URL** |
| **`API`** | Step 2でコピーした **Gemini APIキー** |

※ 名前は必ず大文字で、間違いなく入力してください！

### Step 4. 検索キーワードを決める（Variablesの設定）
「どんなニュースを集めるか」を決めます。

1.  同じ画面（Settings > Secrets and variables > Actions）で、今度は **「Variables」** タブをクリックします。
2.  **「New repository variable」** をクリック。

| Name（名前） | Value（中身） | 説明 |
| :--- | :--- | :--- |
| **`SEARCH_KEYWORDS`** | `AI, 政治, 経済, ポケモン, 推しの名前` | **【日本語OK】** ここにあなたの好きなキーワードを自由に設定してください（カンマ区切り） |
| **`FILTER_KEYWORDS`** | `政治, 事故, crypto` | 除外したいキーワード（カンマ区切り） |

> **📷 [ここにVariables登録画面のスクショを入れる]**

### Step 5. 起動テスト

これで準備完了です！実際に動かしてみましょう。

1.  タブ **「Actions」** をクリック。
2.  左側の **「News Agent」** をクリック。
3.  右側の **「Run workflow」** ボタンを押し、緑のボタンをクリック。
4.  数十秒〜1分ほど待つと、Discordに通知が届きます！🥳

> **📷 [ここにActionsのRun workflowボタンと、Discord通知完了のスクショを入れる]**

## ⚙️ カスタマイズ（上級者向け）

AIの「性格（選定基準）」を変えたい場合は、以下のファイルを編集してください。

*   ファイル: `summarizer.py`
*   場所: 34行目付近の `prompt = ...` の部分

ここの日本語の指示文を書き換えることで、「辛口で要約して」「小学生にもわかるように」「プログラミングコードが含まれる記事だけ選んで」など、自由にカスタマイズ可能です。

---

## ⚠️ 注意事項・トラブルシューティング

*   **通知が来ない場合**:
    *   Actionsタブを見て、全部「緑のチェック✅」になっていますか？ 「赤の×」ならエラーログを確認してください。
    *   Discord Webhook URLの末尾にスペースが入っていませんか？
*   **無料枠について**:
    *   Gemini APIの無料枠は非常に大きいので、個人利用で超えることはまずありませんが、過度なリクエストには注意してください。
